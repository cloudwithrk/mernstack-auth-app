
# Full Auth Task App

## Backend Setup
1. Run `npm install` in the `backend` folder.
2. Copy `.env.example` to `.env` and set your JWT secret.
3. Import `schema.sql` to MySQL.
4. Run `node server.js`.

## Frontend Setup
1. Run `npm install` in the `frontend` folder.
2. Run `npm start`.

## Default ports:
- Backend: 5000
- Frontend: 3000
